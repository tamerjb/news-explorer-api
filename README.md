# News Explorer API

## Technologies

- Node.js
- Express.js
- MongoDB
- Mongoose
- Celebrate/Joi
- Jsonwebtoken

## Routes

- GET /users/me - get user data
- POST /signup - create user
- POST /signin - login user

- GET /articles - get saved articles
- POST /articles - save article
- DELETE /articles/articleId - delete article
